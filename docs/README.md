# SynthFHIR documentation

SynthFHIR is a locally hosted generator of synthetic, fully relational FHIR
patient data. It ships a REST API, a command-line tool, a Python library, and a
web console. It never contacts external services and stores no data.

## A note on synthetic data

Every record SynthFHIR produces is fabricated. Names, identifiers, and clinical
values are generated by rules and random draws and do not describe real people.
The data is intended for development, testing, demos, and education. It is not
suitable for clinical use or for training models that will be used on real
patients without appropriate review.
